# 抖音 6 路由模型

> 针对中文教学内容的 6 路由模型。SKILL.md 的扩展参考。

## 6 路由规则表

| 路由 | 抖音适用内容 | 切分单位 | 边界规则 |
|------|------------|---------|---------|
| `lesson` | 教学课、知识分享、口播讲解 | 每个完整知识点/例句解析/技巧 | 在知识点切换、例句开始、作业布置处切分 |
| `explainer` | 科普解读、论文解析、干货总结 | 完整论点/例子/视觉功能 | 在论点完成+清晰停顿处切分 |
| `conversation` | 访谈、对话、问答、直播切片 | 完整问答/话题单元 | 不按每次发言切，按完整话题切 |
| `demo` | 教程演示、产品使用、操作指南 | 操作步骤+可观察结果 | 保留前置条件和结果 |
| `narrative` | Vlog、故事讲述、情感朗读、剧情 | 叙事段落/情节转换 | 在场景/时间/情绪转换处切分 |
| `bulletin` | 资讯速览、新闻摘要、干货清单 | 资讯条目/要点 | 每条资讯独立成段 |

## 路由判断启发式

```
标题含"第X讲/课程/教学/技巧/速通/精讲/节课" → lesson
标题含"解读/分析/原理/为什么/揭秘/拆解" → explainer
标题含"访谈/对话/问答/采访/连线/直播" → conversation
标题含"教程/演示/操作/怎么用/手把手/实操" → demo
标题含"vlog/Vlog/日常/故事/记录/一日/旅行" → narrative
标题含"速览/盘点/清单/新闻/资讯/汇总" → bulletin
不确定 → 默认 explainer
```

## 各路由的边界规则详解

### `lesson`（教学课）

**适用内容**：英语课、编程课、考研课、知识分享口播

**切分单位**：每个完整知识点 / 例句解析 / 技巧讲解

**边界规则**：
- 在知识点切换处切分（如"接下来我们看第二个原则"）
- 在例句开始处切分（如"我们来看这个例句"）
- 在作业布置处切分
- 一个完整的教学单元为一个场景
- 不要按固定时长切（即使某段超过 5 分钟，只要是一个知识点就不切）

**示例**（英语长难句课程）：
```
## 01. 课程介绍（00:00-00:45）
## 02. 核心原则：一个句子只有一个核心动词（00:45-02:30）
## 03. 例句1：Most economists believe...（02:30-04:15）
## 04. 第二原则：能体现时态和人称变化（04:15-06:00）
## 05. 进阶例句：The decision to quit his job...（06:00-09:30）
## 06. 词汇精讲（09:30-11:00）
## 07. 进阶例句：published 还是 reviewed（11:00-13:30）
## 08. 作业布置（13:30-结尾）
```

### `explainer`（科普解读）

**适用内容**：科普解读、论文解析、干货总结

**切分单位**：完整论点 / 例子 / 视觉功能

**边界规则**：
- 在论点完成 + 清晰停顿处切分
- 保留完整论证链，不要在论证中途切
- 一个论点 + 其支撑例子为一个场景

### `conversation`（访谈对话）

**适用内容**：访谈、对话、问答、直播切片

**切分单位**：完整问答 / 话题单元

**边界规则**：
- **不按每次发言切**（这是反模式）
- 按完整话题切：一个问答或话题单元为一个场景
- 话题转换处切分（如"我们换个话题聊..."）

### `demo`（操作演示）

**适用内容**：教程演示、产品使用、操作指南

**切分单位**：操作步骤 + 可观察结果

**边界规则**：
- 保留前置条件（"先打开设置..."）和结果（"这样就完成了"）
- 一个操作步骤 + 其结果为一个场景
- 在步骤切换处切分

### `narrative`（叙事讲述）

**适用内容**：Vlog、故事讲述、情感朗读、剧情

**切分单位**：叙事段落 / 情节转换

**边界规则**：
- 在场景/时间/情绪转换处切分
- 保留叙事完整性
- 不要在叙事高潮中途切

### `bulletin`（资讯速览）

**适用内容**：资讯速览、新闻摘要、干货清单

**切分单位**：资讯条目 / 要点

**边界规则**：
- 每条资讯独立成段
- 不要合并多条资讯
- 在"下一条..."或编号切换处切分

## 路由组合（未来考虑）

v1 暂不支持路由组合，主对话选一个最匹配的路由即可。未来可考虑：
- `lesson + narrative`：故事化教学
- `conversation + bulletin`：访谈中穿插资讯条目
